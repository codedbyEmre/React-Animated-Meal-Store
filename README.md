# React Animated Meal Store

## Stacks

```
React 18
React Router 6
Framer Motion
Tailwind
Daisy UI
```

The API used on this project can be found <a href="https://www.themealdb.com/" target="_blank">here</a>

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

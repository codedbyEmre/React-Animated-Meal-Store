# React Animated Meal Store

## Stacks

```
React 18
React Router 6
Framer Motion
Tailwind
Daisy UI
Vite
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
